GitHub Integration

MrHyde lets you access your GitHub account and browse your repositories on the go, create and edit new files and push all those changes back to GitHub. You can even work with private and organization repositories.

Have a big repo that you don't / cannot checkout via git on your phone. No problem! MrHyde only downloads those files that you need for editing and keeps everything else on GitHub. So you get the best of both worlds, access to all your files while not filling up precious memory on your phone.

Full preview

So now that you have written that awesome post that you have always wanted the world to see, wonder what it looks like with all your custom CSS and templates? Good thing MrHyde has a preview function built in!

Whenever you want to preview changes to a post, a diff of your local changes is sent to our webservices, which create a temporary clone of your repository, apply the diff, compile the project using a full Jekyll installation and return a preview URL which you can view like any other webpage.

Worried about other seeing your unfinished post before publishing? No need. Preview URLs are randomly generated and auto-deleted 10 minutes after first creating them, making sure that we don't steal any of your spotlight when finally releasing your post to the world.
